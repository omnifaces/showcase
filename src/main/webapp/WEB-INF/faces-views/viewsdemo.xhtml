Viewsdemo

This page resides in the /WEB-INF/faces-views folder as viewsdemo.xhtml and has been automatically scanned by FacesViews. It can be requested with both its original extension (viewsdemo.xhtml) as well as without its extension (viewsdemo). When requesting with an extension a redirect will take place to its extensionless variant.

To demo that extensionless URLs can be used in combination with other URL rewriters (like the CDI conversation scope), the PostRedirectGet variant starts a conversation. When going back and forth between the demo pages with PRG, the created time should stay the same after the first navigation.
