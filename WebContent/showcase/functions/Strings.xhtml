Collection of EL functions for string manipulation: of:abbreviate(), of:capitalize(), of:concat(), of:prettyURL(), of:encodeURL(), of:escapeJS() and of:formatX().

Instead of of:formatX(), you can also use .

Abbreviate a long string: #{of:abbreviate(string1, 20)}

Capitalize a string: #{of:capitalize(string2)}

Concat 2 strings: #{of:concat(string2, string3)}

Following functions deal with the following string: "#{string4}"

  • Create a pretty URL (part): #{of:prettyURL(string4)} check link URL (don't click :) )
  • URL-encode with UTF-8: #{of:encodeURL(string4)} click link
  • Escape for usage in JavaScript: #{of:escapeJS(string4)} click to see alert

Format strings in EL:

  • #{of:format1('one {0}', 'two')}
  • #{of:format2('one {0} three {1}', 'two', 'four')}
  • #{of:format3('one {0} three {1} five {2}', 'two', 'four', 'six')}
  • #{of:format4('one {0} three {1} five {2} seven {3}', 'two', 'four', 'six', 'eight')}
  • #{of:format5('one {0} three {1} five {2} seven {3} nine {4}', 'two', 'four', 'six', 'eight', 'ten')}
