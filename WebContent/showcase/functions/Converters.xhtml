Collection of EL functions for data conversion: of:formatBytes(), of:setToList(), of:mapToList() and of:printStackTrace().

Notes:

  • Since JSF 2.2 it should be possible to use Set<E> and Map<K, V> in <ui:repeat>. The \#{of:setToList} and \#{of:mapToList} are then only useful for JSF 2.0 and 2.1.
  • There's also an \#{of:printStackTrace}. See it in action in page.

Format a given amount of bytes with IEC binary unit: #{of:formatBytes(1023)}, #{of:formatBytes(1024)}, #{of:formatBytes(500000)}, #{of:formatBytes(1048576)}

Convert Set<E> to List<E>: #{item}#{!loop.last ? ', ' : ''}

Convert Map<K, V> to List<Entry<K, V>>: #{entry.key}=#{entry.value}#{!loop.last ? ', ' : ''}
