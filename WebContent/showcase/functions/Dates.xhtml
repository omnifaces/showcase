Collection of EL functions for date and time: of:formatDate(), of:formatDateWithTimezone(), of:addXxx() like of:addDays(), of:xxxBetween() like of:daysBetween(), of:getMonths(), of:getShortMonths(), of:getDaysOfWeek() and of:getShortDaysOfWeek().

Format current date with system default timezone: #{of:formatDate(now, 'yyyy-MM-dd HH:mm:ss')}

Format current date with GMT timezone: #{of:formatDateWithTimezone(now, 'yyyy-MM-dd HH:mm:ss', 'GMT')}

Add 3 months to today and format it with system default timezone: #{of:formatDate(threeMonthsLater, 'yyyy-MM-dd HH:mm:ss')}

Count months between the date representing "now" and the date representing "3 months later": #{of:monthsBetween(now, threeMonthsLater)} months
